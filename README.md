## 🤔 **为什么要开源？**  
1. **自我监督**：公开了就不能偷懒了😅。  
2. **积少成多**：碎片知识也是知识，万一帮到未来的自己（或你）呢？  
3. **交个朋友**：如果你发现错误，欢迎 **提 Issue** 或 **PR**，我们一起进步！  

## 🎉 **使用指南**  
- **随便看**：像逛超市一样自由，但不用付钱💰。  
- **别全信**：有些笔记可能是半夜写的，不保证清醒程度☕。  
- **笑一笑**：如果看到幼稚的代码，那一定是我早期作品（理直气壮.jpg）。
